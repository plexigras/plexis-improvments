Plexi's Improvments
==========

A improved version of the default Minecraft resource pack based of of Plastix's [Plast Pack](https://github.com/Plastix/Plast-Pack).



[Download](https://github.com/plexigras/plexis-improvments/archive/master.zip)
---

Containing
---
* 749 textures
  * 496 block textures
  * 230 item textures
  * 53 mob textures
  * trasparent gui
  * icons and particles
  * armor textures
  * paintings
  * and more
* 140 sound files
* 86 fixed default models
  * 58 block models
  * 27 item models
* optifine/mcpatcher
  * connected textures
  * custom colors
  * natural textures
  * random mobs
* custom language file
* custom font
