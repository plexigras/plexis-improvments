Plexi's Improvments
==========

A improved version of the default Minecraft resource pack based of of Plastix's [Plast Pack](https://github.com/Plastix/Plast-Pack).



[Download](https://github.com/plexigras/plexis-improvments/archive/master.zip)
---

Containing
---
* 628 textures
  * 293 block textures
  * 230 item textures
  * 42 mob textures
  * trasparent gui
  * icons and particles
  * armor textures
  * paintings
  * and more
* 140 sound files
* 44 fixed default models
  * 40 block models
  * 4 item models
* optifine/mcpatcher
  * connected textures
  * custom colors
  * natural textures
* custom language file
* custom font
